Importante para os PDFs funcionarem no GitHub

Os arquivos dentro da pasta pdfs precisam ter exatamente o mesmo nome usado nos botões do site, incluindo hífens, letras minúsculas e a extensão .pdf.

Exemplo: ./pdfs/informatica-criptografia-e-backup.pdf
